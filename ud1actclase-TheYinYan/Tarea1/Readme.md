# **Parte 2: Comparación y reflexión**

- **¿Qué lenguajes tienen entornos de desarrollo más variados o flexibles?**

    Los lenguajes más flexibles en cuanto a entornos son Python, JavaScript, Java y C++, porque tienen soporte en múltiples IDEs y editores, además de amplias áreas de aplicación.

- **¿Cuál parece tener más aplicaciones en empresas globales?**

    El lenguaje con más aplicaciones en empresas globales es Java, seguido muy de cerca por Python y JavaScript.

    - **Java:** fuerte en aplicaciones corporativas grandes.
    
    - **Python:** fuerte en ciencia de datos y automatización.

    - **JavaScript:** esencial para cualquier desarrollo web moderno.

- **¿Crees que la popularidad en TIOBE se relaciona con el uso en grandes compañías?**

    La popularidad en TIOBE **sí refleja** bastante el uso en grandes empresas, especialmente para lenguajes generalistas como Java, Python y JavaScript. Pero hay excepciones: algunos lenguajes muy usados en nichos corporativos no aparecen tan alto porque TIOBE mide globalidad, no especialización.

- **¿Qué lenguaje te gustaría explorar más y por qué?**

    Aparte de Java por que ya lo estoy viendo en otro modulo me gustaria aprender mas de Phython y JavaScript.
