# Descripción de la práctica

Utiliza el diagrama de clases creado en la práctica anterior para implementar el software que modela.

# Codigo generado

## 📁Estructura del proyecto

```
Practica 2/
    ├── 📄 readme.md
    ├── src/
    ├── ♨️ CPIFP.java
    ├── ♨️ App.java
    └── ♨️ Estudiante.java
```

- [App](./src/App.java): Clase principal que contiene el método `main` y gestiona el menú de la aplicación.

- [Class CPIFP](./src/CPIFP.java): Clase principal que representa el centro educativo, con métodos para gestionar estudiantes, departamentos y obtener información.

- [Class Estudiante](./src/Estudiante.java): Clase que representa a un estudiante, con atributos para el nombre y el identificador.
